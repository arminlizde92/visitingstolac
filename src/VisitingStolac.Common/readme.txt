VisitingStolac.Common readme.txt

Folders:
DTO: Basically folder which holds all the subfolders for DTOs
    Subfolders: Every entity needs to have a folder for the entities dtos. In the DTO should be all the validation
                for the entity.
Enumerations:   Folder for enumerations
Exceptions: Folder for custom exceptions
Extensions: Folder for custom extensions
Helpers:    Folder for helper classes
Snippets:   Folder for the implementation of useful snippets. After the creation of a new snippet it has to be added into
            VS by clicking Tools => Snipper Manager
